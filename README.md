# Ivy League Facts

## Overview

Ivy League Facts is a Python desktop application built with Tkinter that allows users to explore information about all eight Ivy League universities. The application provides quick access to key facts including location, founding year, academic strengths, and what each school is best known for.

This project was created as a beginner-friendly Python GUI application and serves as the foundation for a future Ivy League recommendation system.

---

## Features

- View information for all 8 Ivy League schools
- Simple desktop graphical user interface (GUI)
- Built entirely with Python and Tkinter
- Displays:
  - School Name
  - Location
  - Founding Year
  - Academic Strengths
  - Best Known For

---
